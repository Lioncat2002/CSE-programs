# CSE-programs
 All the programs we have done till date in VIT

## How to run?

#### Odin programs:

Make sure that you have odin installed <br>
to run:<br>
- `odin run <file_name>.odin`

#### Python Programs:

Make sure that you have `python 3.5` or higher installed<br>
to run:<br>
- `python <filename>.py`
